# 7番天の川
こちらは”7番天の川”のUnityプロジェクトです。
ゲームを実行するときは/Assets/Geek/Scenes/01_Titleシーンからお願いします。　

# Unityroomにて配信中
[こちら](https://unityroom.com/games/amanogawa-7)からブラウザ上でプレイできます。

# 操作方法
## キャラクター
前進：Wキー, 上キー

後退：Sキー, 下キー

右に進む：Dキー, 右キー

左に進む：Aキー, 左キー

視点移動：マウスを移動
## メニュー
BGM,SE音量調整、マウス感度調整メニューの表示非表示：Escapeキー
